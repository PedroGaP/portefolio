ExpenseTracker

Plataforma de gestão de despesas pessoais

Descrição

A ExpenseTracker é uma plataforma criada com o objetivo de ajudar os utilizadores a gerirem as suas despesas pessoais de forma eficiente e organizada. Com uma interface intuitiva e funcionalidades práticas, a aplicação permite aos utilizadores registar, categorizar e analisar as suas despesas diárias, semanais e mensais.

Tecnologias

  • HTML
  • CSS
  • JavaScript
  • LocalStorage

Funcionalidades

Funcionalidade Descrição
Registo de despesas Adicionar despesas diárias com valor e descrição.
Categorização Atribuir categorias como Alimentação, Transporte, Lazer, etc.
Resumo automático Visualização de totais por dia, semana e mês.
Armazenamento local Os dados são guardados no LocalStorage do navegador.
